# FF14 穩定金策分析工具

這是一個自動化市場分析程式，能從 Universalis 抓取商品數據，並計算出利潤與流動性兼具的優質商品。

## 🛠️ 初次使用說明

在第一次執行程式前，請開啟終端機並運行以下指令來安裝必要的環境套件：

    pip install -r requirements.txt

## 🚀 執行方式

安裝完套件後，直接執行主程式即可：

    python main.py

## 📌 功能特點
1. 自動化抓取：利用 Thread Pool 並行處理技術，大幅提升資料下載速度。
2. 智能篩選：整合「成交頻率、價格穩定度、成交量」等多項指標進行加權評分。

## ⚠️ 注意事項
- 玩家巔峰時段（如台灣時間晚上）可能導致分析速度明顯下降或連線不穩。
